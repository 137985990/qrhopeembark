# qrhopeembark

PHP + MySQL QR redirect and intake project for Hope Embark.

## Included
- Unified login for admin and counselors
- QR landing form with counselor attribution
- Monthly reports and counselor dashboards
- Admin management pages
- CSV export for submissions

## Setup
1. Copy `config.example.php` to `config.php`
2. Fill in database and admin credentials
3. Import `sql/schema.sql`
4. Serve the project from a PHP-enabled web root

## Important
- `config.php` is intentionally excluded from git
- The repository mirrors the latest live deployment state
